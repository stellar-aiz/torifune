# torifune

[![CI](https://github.com/stellar-aiz/torifune/actions/workflows/ci.yml/badge.svg)](https://github.com/stellar-aiz/torifune/actions/workflows/ci.yml)
[![Release](https://github.com/stellar-aiz/torifune/actions/workflows/release.yml/badge.svg)](https://github.com/stellar-aiz/torifune/releases)

レシートや領収書をparseし、集計するデスクトップアプリです.

# Development

Prerequisites

- `pnpm`
- `cargo`

Getting Started

- `pnpm install`
- `pnpm tauri dev`
